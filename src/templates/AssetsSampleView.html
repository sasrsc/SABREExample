Assets may be embedded in one of several ways. These ways are described below.

Using `${context.getModule().getManifest().url}/assets/sabre-logo-white.svg` passed via model


Using <img src="modules/com-sabre-example-redapp-web-module/assets/mm.jpg" /> directly in template


Using <img src="com-sabre-example-redapp-web-module/assets/mm.jpg" /> directly in template


Using dash-style CSS Variable: <img style="content: var(--modules--com-sabre-example-redapp-web-module--assets--mm_jpg)" />


Using dash-style shortened CSS Variable: <img style="content: var(--com-sabre-example-redapp-web-module--assets--mm_jpg)" />


Using path-style CSS Variable: <img style="content: var(--modules\/com-com-sabre-example-redapp-web-module\/assets\/mm\.jpg)" />


Using path-style shortened CSS Variable: <img style="content: var(--com-sabre-example-redapp-web-module-web-module\/assets\/mm\.jpg)" />
